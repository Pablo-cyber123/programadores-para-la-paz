Al enviar una petición POST a la ruta /registro con el JSON:
{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor recibe estos datos a través de req.body y responde con el siguiente objeto JSON estructurado, confirmando que la información fue procesada correctamente:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}
