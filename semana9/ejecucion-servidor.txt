- Qué ocurre cuando ejecutas node server.js: 
Inicia el proceso de Node que lee el archivo y levanta el servidor web en tu máquina.

- Qué mensaje aparece en la terminal: 
Aparece el log que definimos en el código: "Servidor ejecutándose en puerto 3000".

- Qué sucede cuando visitas la ruta /estado: 
El servidor recibe la petición HTTP GET y responde enviando al navegador un objeto JSON con el estado "Servidor funcionando" y el servicio "API comunitaria".
